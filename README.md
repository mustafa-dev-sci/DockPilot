## DockPilot
From docking link to clean research report.

DockPilot is a lightweight research tool that turns HDOCK docking results into organized tables, renamed model files, screenshots, reports, and shareable project packages.

Features:
• Collect public HDOCK result pages
• Extract clean docking score tables
• Download and rename model files
• Capture 3D result screenshots
• Generate PDF and HTML reports
• Export shareable project packages
